### Next.js
A TypeScript starter template for Next.js that includes:
- 🔵 Typescript
- 💖 Prettier
- ⚛️ Redux Toolkit
- 🔐 React Auth kit
- 🐶 Husky
- 🎨 Tailwind
- 🗂️ `@/` alias import
- ⚡️ SWR 
- 🟣 Axios
- 🐞 Jest
- 🌲 Cypress

### 📦 install
```
npm install
```

### ⚡️ development
```
npm run dev
```

### 🚀 Build
```
npm run build
```

### 🐞 Testing
```
npm run test
```
